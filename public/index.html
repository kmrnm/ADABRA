ADABRA

Real-time classroom buzzer system inspired by “Brain Ring”. Paper-first: the host reads questions from paper, ADABRA handles timing, buzzing, and scoring.

Host a game
Tip: the host will share a room code or link. No sign-up required for players.
1) Host creates a room
Share the player link or room code with teams.
2) Teams join and select team
Team selection is locked per device (refresh-safe).
3) Beep starts the round
First buzz locks the round; host marks correct/incorrect.
